
## Gevraagd
Schrijf een functie `moestuin(tuin1, tuin2)` die gegeven twee moestuinen controleert of deze aan dezelfde regels voldoen.


#### Voorbeeld

```python
>>> moestuin([["🍅", "🥕", "🥕", "🥦"],
              ["🍅", "🥕", "🥕", "🥦"],
              ["🍅", "🍅", "🥕", "🧄"],
              ["🍅", "🍅", "🥕", "🧄"]],
             [["🥦", "🥕", "🍅"],
              ["🥦", "🥕", "🍅"],
              ["🧄", "🥕", "🍅"]])
True
```
