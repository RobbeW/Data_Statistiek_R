## Gevraagd
Schrijf een functie `transponeer(matrix)` die een gegeven matrix **transponeert**.

#### Voorbeeld

```python
>>> transponeer([[1, 2, 3],
                 [4, 5, 6]])
[[1, 4],
 [2, 5],
 [3, 6]]
```
