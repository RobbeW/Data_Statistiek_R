## Gevraagd
Schrijf een functie `aantal_ballonnen(woord)` dat gegeven een woord, het maximale aantal keer dat het woord ballon kan gevormd worden met de letters, bepaalt.

Bestudeer grondig onderstaande voorbeelden.

#### Voorbeelden
